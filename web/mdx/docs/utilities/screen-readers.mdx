The `Screen reader` utility is usually used when content should be made available to screen reader users but hidden from sighted users.

The table below describe how the `screen-reader` utility is applied

## Types

          border-width: 0; <br />
          clip: rect(0, 0, 0, 0); <br />
          height: 1px; <br />
          margin: -1px; <br />
          overflow: hidden; <br />
          padding: 0; <br />
          position: absolute; <br />
          white-space: nowrap; <br />
          width: 1px;
        </kbd>
      </td>
    </tr>
  </tbody>
</table>

## Props

This utility can also be used on targeted breakpoints, e.g. use `md-sr-only` to apply the `sr-only` only on a medium device and above.

```html
<div class="md-sr-only">
  .   .   .
</div>
```
