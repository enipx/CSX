## Usage

```html rose
<template preview>
  <div class="ta-center fz-sm mb-4">
    <p class="fw-600 op-70 c-pink-700">
      .fx-row
    </p>
    <p class="c-secondary-500 fw-500 px-5">
      This is the default direction and set the flex items horizontally and its same as text direction.
    </p>
  </div>

  <div class="d-flex fxd-row fx-center bd-brand bdw-3 p-2 c-secondary-700">
    <div class="mx-2 ta-center fw-600 w-20 h-12 fx-center bg-purple-100 bd-purple-200">
      1
    </div>

    <div class="mx-2 ta-center fw-600 w-20 h-12 fx-center bg-purple-100 bd-purple-200">
      2
    </div>

    <div class="mx-2 ta-center fw-600 w-20 h-12 fx-center bg-purple-100 bd-purple-200">
      3
    </div>
  </div>
</template>

<div class="d-flex fxd-row ...">
  <div class="w-20 h-10 ...">1</div>
  <div class="w-20 h-10 ...">2</div>
  <div class="w-20 h-10 ...">3</div>
</div>
```
