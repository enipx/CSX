## Usage

```html rose
<template preview>
  <div>
    <p class="fz-sm ta-center fw-600 c-pink-700 op-70 tt-uppercase mb-4">Visibility: visible</p>
    
    <div class="fx-center">
      <div class="mx-2 ta-center fw-600 w-20 h-20 fx-center bg-purple-100 bd-purple-200">
        1
      </div>

      <div class="mx-2 vis-visible ta-center fw-600 w-20 h-20 fx-center bg-purple-100 bd-purple-200 bdw-3">
        2
      </div>

      <div class="mx-2 ta-center fw-600 w-20 h-20 fx-center bg-purple-100 bd-purple-200">
        3
      </div>
    </div>
  </div>
</template>

<div>
  <div class="fx-center ...">
    <div class="w-20 h-20 ...">1</div>
    <div class="vis-visible w-20 h-20 ...">2</div>
    <div class="w-20 h-20 ...">3</div>
  </div>
</div>
```
