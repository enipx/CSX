  description="sets the <code>max-width</code> property of an element to match the full width of the selected breakpoint when they reach their breakpoint. In case no breakpoint is specified, the <code>width</code> property is set to <strong>100%</strong>"
/>

**NOTE:** By Default, The `Container` is automatically center horizontally in the <a target="_blank" href="https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model/Introduction">DOM</a> when they do not take up the full width of the parent.
