## Usage

```html rose
<template preview>
  <div>
    <p className="fz-sm ta-center fw-600 op-70 tt-uppercase c-pink-700">Content Box</p>
    <div className="fx-center pb-16 my-4">
      <div className="bd-brand bg-pink-50 bdw-5 w-60 h-20 fw-600">
        <p className="ta-center">Parent</p>
        <div className="bdw-5 bd-purple-400 bg-purple-50 p-4 bxz-content w-full h-full fx-center">
          Child
        </div>
      </div>
    </div>
    <p className="fz-sm ta-center fw-500 op-70">The Child element has a padding of 1rem (16px) and border width of 5px</p>
  </div>
</template>

    ...
  </div>
</div>
```
