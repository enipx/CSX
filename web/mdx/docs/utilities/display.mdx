<DocsTable
  name="display"
  description="allows us to define how an element is displayed as a block or inline element. The default display value for the majority of elements is block or inline."
/>

## Usage

```html rose
<template preview>
  <!-- **---------** Block **--------** -->
  <div class="">
    <div class="bd-brand bg-pink-50 bdw-5 fw-500 p-2 fz-sm c-secondary-700">
      <div class="d-block bd-purple-200 bg-purple-100 h-12 fx-center  mb-2">
        display: block
      </div>
      <div class="d-block bd-purple-200 bg-purple-100 h-12 fx-center  mb-2">
        display: block
      </div>
      <div class="d-block bd-purple-200 bg-purple-100 h-12 fx-center">
        display: block
      </div>
    </div>
  </div>
</template>

<div class="bd-brand bdw-5 ...">
  <div class="d-block ...">block</div>
  <div class="d-block ...">block</div>
  <div class="d-block ...">block</div>
</div>
```
