---
title: Object Fit
description: Used to set how an element like img or video must be resized to fit its parent element.
---

<DocsTable
  name="object-fit"
  head={[
    'Name',
    'Properties',
    'Description'
  ]}
  description="set how an element needs to be resize to fit the parent element."
  content={{
    contain: 'Used to resized an element to contain it parent element.',
    cover: 'Used to resized an element to cover while maintaining its aspect ratio of its parent element.',
    fill: 'Used to resized an element to fill it parent element.',
    none: 'Used to set element to not be resized in its parent element.',
    'scale-down': 'Used to set element to not be resized in its parent element but scale down if neccessary to fit. this behaves as if <code>obj-none</code> or <code>obj-contain</code> were specified.',
  }}
  hideDataClass
/>

## Usage

```html rose
<template preview>
  <div>
    <p class="fz-sm ta-center fw-600 c-pink-700 op-70 tt-uppercase my-4">Object-Fit: Contain</p>
    <div class=" w-80 h-56 bg-purple-100 mx-auto">
      <img class="obf-contain w-full h-full" src="https://placeimg.com/200/100/arch" />
    </div>
  </div>
</template>

<div class=" w-80 h-56 bg-purple-100">
  <img class="obf-contain w-full h-full" src="https://placeimg.com/200/100/arch" />
</div>
```
