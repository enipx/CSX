## Usage

```html
<template preview>
  <div>
    <input type="text" class="input" placeholder="Enter name"/>
  </div>
</template>

<div>
  <input type="text" class="input" placeholder="Enter name" />
</div>
```

## Sizings
Set input size using the `.input-[size]` utility class or `data-size="[size]"` attribute. Options available are `sm`, `md`, `lg`. Usage example is `input-lg` or `data-size="lg"`.

```html
<template preview>
  <div class="mb-2">
    <input type="text" class="input input-sm" placeholder="Enter name" />
  </div>

  <div class="mb-2">
    <input type="text" class="input input-md" placeholder="Enter name" />
  </div>

  <div class="mb-2">
    <input type="text" class="input input-lg" placeholder="Enter name" />
  </div>
</template>

<div>
  <input type="text" class="input" placeholder="Enter name" data-size="sm" />
  <input type="text" class="input" placeholder="Enter name" data-size="md" />
  <input type="text" class="input" placeholder="Enter name" data-size="lg" />
</div>
```

## Styles
Update input appearance using the `.input-[style]` utility class or `data-style="[style]"` attribute. Options available are `flushed`, `brick`, `unstyled`. Usage example is `input-brick` or `data-style="brick"`.

```html transparent
<template preview>

  <div class="mb-2">
    <input type="text" class="input" placeholder="Enter name" />
  </div>

  <div class="mb-2">
    <input type="text" class="input" placeholder="Enter name" data-style="brick" />
  </div>

  <div class="mb-2">
    <input type="text" class="input" placeholder="Enter name" data-style="flushed" />
  </div>

  <div class="mb-2">
    <input type="text" class="input" placeholder="Enter name" data-style="unstyled" />
  </div>

</template>

<div>
  <input type="text" class="input" placeholder="Enter name" />
  <input type="text" class="input" placeholder="Enter name" data-style="brick" />
  <input type="text" class="input" placeholder="Enter name" data-style="flushed" />
  <input type="text" class="input" placeholder="Enter name" data-style="unstyled" />
</div>
```

## States
Update input state by using 
 - `.input-valid` class or `data-valid`/`aria-invalid="false"` attribute to indicate a valid state,
 - `.input-invalid` class or `data-invalid`/`aria-invalid="true"` attribute to indicate an invalid state, 
 - `disabled` attribute to indicate a disabled state,
 - `readonly`, attribute to indicate a readonly state.

```html transparent
<template preview>
  <div class="mb-2">
    <input type="text" class="input" placeholder="Enter name" data-invalid />
  </div>

  <div class="mb-2">
    <input type="text" class="input" placeholder="Enter name" data-valid />
  </div>

  <div class="mb-2">
    <input type="text" class="input" placeholder="Disabled" disabled />
  </div>

  <div class="input-icon mb-2">
    <input type="text" class="input input-icon_input" placeholder="Readonly" readonly />
    <span class="input-icon_icon c-dark">
      <svg fill="none" viewBox="0 0 20 20" xmlns="http://www.w3.org/2000/svg" width="18">
        <g fill="currentColor">
          <path d="m9.9973 12.646c-3.5942 0-6.6642 0.5667-6.6642 2.8333 0 2.2675 3.0508 2.8542 6.6642 2.8542 3.5942 0 6.6642-0.5658 6.6642-2.8333s-3.05-2.8542-6.6642-2.8542z"/>
          <path d="m9.9973 10.486c2.4483 0 4.41-1.9625 4.41-4.41s-1.9617-4.41-4.41-4.41c-2.4475 0-4.41 1.9625-4.41 4.41s1.9625 4.41 4.41 4.41z" opacity=".4"/>
        </g>
      </svg>
    </span>
  </div>
</template>

<div>
  <input type="text" class="input" placeholder="Valid" data-valid />
  <input type="text" class="input" placeholder="Invalid" data-invalid />
  <input type="text" class="input" placeholder="Disabled" disabled />
  <div class="input-icon">
    <input type="text" class="input input-icon_input" placeholder="Readonly" readonly />
    <span class="input-icon_icon">
      . . .
    </span>
  </div>
</div>
```


## Label & Caption
To associate an input with a label or caption, use the label element together with the input and `input-caption` class for the caption. See example below.


```html
<template preview>
  <div class="mb-4">
    <label for="username">Username</label>
    <input class="input" type="text" id="username" />
  </div>

  <div>
    <label for="email" data-required>Email</label>
    <input class="input" type="email" id="email" />
    <p class="input-caption">
      This is attributed to your account.
    </p>
  </div>
</template>

<div>
  <div class="mb-4">
    <label for="username">Username</label>
    <input class="input" type="text" id="username" />
  </div>

  <div>
    <label for="email" data-required>Email</label>
    <input class="input" type="email" id="email" />
    <p class="input-caption">
        This is attributed to your account.
    </p>
  </div>
</div>
```
