---
title: "Avatar"
description: "Used to add options when only a single option can be selected from the given options."
---

## Usage

```html
<template preview>
  <div class="d-flex">
    <div class="custom-avatar m-4" data-size="sm">
      <img src="https://picsum.photos/id/237/200/300
      " alt="avatar">
    </div>

    <div class="custom-avatar m-4" data-size="sm">
      <p class="custom-avatar_name">HS</p>
    </div>
  </div>
</template>

<div>
  <div class="custom-avatar" data-size="sm">
    <img src="https://picsum.photos/id/237/200/300
    " alt="avatar">
  </div>

  <div class="custom-avatar" data-size="sm">
    <p class="custom-avatar_name">HS</p>
  </div>
</div>
```

## Sizings
Set avatar size using the `.avatar-[size]` utility class or `data-size="[size]"` attribute. Options available are `sm`, `md`, `lg`. Usage example is `avatar-lg` or `data-size="lg"`.

```html
<template preview>
  <div class="d-flex">
    <div class="custom-avatar m-4" data-size="sm">
      <img src="https://picsum.photos/id/237/200/300
      " alt="avatar">
    </div>

    <div class="custom-avatar m-4" data-size="md">
      <img src="https://picsum.photos/id/237/200/300
      " alt="avatar">
    </div>

    <div class="custom-avatar m-4" data-size="lg">
      <img src="https://picsum.photos/id/237/200/300
      " alt="avatar">
    </div>
  </div>

  <div class="d-flex">
    <div class="custom-avatar m-4" data-size="sm">
      <p class="custom-avatar_name">HS</p>
    </div>

    <div class="custom-avatar m-4" data-size="md">
      <p class="custom-avatar_name">HS</p>
    </div>

    <div class="custom-avatar m-4" data-size="lg">
      <p class="custom-avatar_name">HS</p>
    </div>
  </div>
</template>
```

## Squared Icons


## Add Notification
Add notification icon to avatar using the `custom-avatar_notification` utility class. You can also set the position of the icon using the `[data-pos="top | bottom"]` attribute.

```html
<template preview>
  <div class="d-flex">
    <div class="custom-avatar m-4" data-size="sm">
      <img src="https://picsum.photos/id/237/200/300
      " alt="avatar">
      <div class="custom-avatar_notification" data-pos="top"></div>
    </div>

    <div class="custom-avatar m-4" data-size="sm">
      <p class="custom-avatar_name">HS</p>
      <div class="custom-avatar_notification" data-pos="top"></div>
    </div>

    <div class="custom-avatar m-4" data-size="sm">
      <img src="https://picsum.photos/id/237/200/300
      " alt="avatar">
      <div class="custom-avatar_notification" data-pos="bottom"></div>
    </div>

    <div class="custom-avatar m-4" data-size="sm">
      <p class="custom-avatar_name">HS</p>
      <div class="custom-avatar_notification" data-pos="bottom"></div>
    </div>
  </div>

  <div class="d-flex">
    <div class="custom-avatar m-4" data-size="sm" data-squared>
      <img src="https://picsum.photos/id/237/200/300
      " alt="avatar">
      <div class="custom-avatar_notification" data-pos="top"></div>
    </div>

    <div class="custom-avatar m-4" data-size="sm" data-squared>

    </div>

      " alt="avatar">

      <p class="custom-avatar_name">HS</p>
      <div class="custom-avatar_notification" data-pos="bottom"></div>
    </div>
  </div>
</template>
```
