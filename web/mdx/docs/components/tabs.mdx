## Usage


```html
<template preview>
  <div class="tabs mb-4">
    <div class="tab_nav" data-variant="" data-color=''>
      <button class="tab_nav_item active" data-toggle="tabs" data-target="#tab1">Tab 1</button>
      <button class="tab_nav_item" data-toggle="tabs" data-target="#tab2">Tab 2</button>
    </div>

    <div class="tab_content">
      <div class="tab_panel active" id="tab1" role="tabpanel">Tab 1 Content...</div>
      <div class="tab_panel" id="tab2" role="tabpanel">Tab 2 Content...</div>
    </div>
  </div>
</template>
```

## Variants

Set tabs variants using `data-variant="fenced | round"`. Options available are `fenced`, `round`. Usage example is `data-variant="round"`.


## Tab Nav Item Color
Set tabs nav color using `data-color="primary | success | danger"`. Options available are `primary`, `success`, `danger`. Usage example is `data-color="primary"`.

```html transparent
    <div class="tab_nav" data-variant="round" data-color='primary'>
```
