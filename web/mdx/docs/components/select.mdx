## Usage

```html
<template preview>
  <div>
    <select class="select">
      <option>Select Option</option>
      <option>Option 1</option>
      <option>Option 2</option>
      <option>Option 3</option>
    </select>
  </div>
</template>

<div>
  <select class="select">
    <option>Select Option</option>
    . . .
  </select>
</div>
```

## Sizings
Set select size using the `.select-[size]` utility class or `data-size="[size]"` attribute. Options available are `sm`, `md`, `lg`. Usage example is `select-lg` or `data-size="lg"`.

```html
<template preview>
  <div class="mb-2">
    <select class="select select-sm">
      <option>Select Option</option>
      <option>Option 1</option>
      <option>Option 2</option>
      <option>Option 3</option>
    </select>
  </div>

  <div class="mb-2">
    <select class="select select-md">
      <option>Select Option</option>
      <option>Option 1</option>
      <option>Option 2</option>
      <option>Option 3</option>
    </select>

      <option>Select Option</option>
```

## States
Update select state by using 
 - `.select-valid` class or `data-valid`/`aria-invalid="false"` attribute to indicate a valid state,
 - `.select-invalid` class or `data-invalid`/`aria-invalid="true"` attribute to indicate an invalid state, 
 - `disabled` attribute to indicate a disabled state,
 - `readonly`, attribute to indicate a readonly state.

```html transparent
<template preview>
  <div class="mb-2">
    <select class="select" data-invalid>
      <option>Select Option</option>
      <option>Option 1</option>
      <option>Option 2</option>
      <option>Option 3</option>
    </select>
  </div>

  <div class="mb-2">
    <select class="select" data-valid>
      <option>Select Option</option>
      <option>Option 1</option>
      <option>Option 2</option>

    </select>
  </div>
```
