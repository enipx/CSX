---
title: "Dropdown"
description: "Used as a toggleable box that allows a user to select a value from a series of options"
---

## Usage

```html
<template preview>
  <div class="dropdown mt-4">
    <button class="btn" data-toggle="dropdown" id="dropdown-toggle-btn">
      Dropdown
    </button>
    <div class="dropdown_menu" data-arow data-align="left" data-hover-reveal>
      <a class="dropdown_menu_header">Snacks</a>
      <a class="dropdown_menu_item">Cheese</a>
      <a class="dropdown_menu_item">Hamburger</a>
      <a class="dropdown_menu_item">Pizza</a>
      <div class="dropdown_divider"></div>
      <div class="dropdown_menu_content">
        <button class="btn btn-secondary w-full">See more..</button>
      </div>
    </div>
  </div>

</template>
```

## Alignment
Set dropdown alignment using the `data-align="[align]"` attribute. Options available are `right`, `center`, `left`. Usage example is `data-align="right"`.
