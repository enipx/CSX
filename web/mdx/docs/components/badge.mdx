---
title: "Badge"
description: "Used to emphasize the state of an item to quickly identify it."
---

## Usage

```html transparent
<template preview>
  <div>
    <div class="badge"> Default</div>
    <div class="badge-success"> Success</div>
    <div class="badge-danger"> Danger</div>
    <div class="badge-primary"> Progress</div>
    <div class="badge-info"> Recommended</div>
    <div class="badge-warning"> Busy</div>
    <div class="badge-attention"> Moral</div>
  </div>
</template>

<div>
  <div class="badge"> Default</div>
  <div class="badge-success"> Success</div>
  <div class="badge-danger"> Danger</div>
  <div class="badge-primary"> Progress</div>
  <div class="badge-info"> Recommended</div>
  <div class="badge-warning"> Busy</div>
  <div class="badge-attention"> Moral</div>
</div>
```
