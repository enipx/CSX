## Usage

```html transparent
<template preview>
  <div>
    <span class="status-hinter"></span>
    <span class="status-hinter-success"></span>
    <span class="status-hinter-danger"></span>
    <span class="status-hinter-primary"></span>
    <span class="status-hinter-info"></span>
    <span class="status-hinter-warning"></span>
    <span class="status-hinter-attention"></div>
  </div>
</template>

<div>
  <span class="status-hinter"></span>
  <span class="status-hinter-success"></span>
  <span class="status-hinter-danger"></span>
  <span class="status-hinter-primary"></span>
  <span class="status-hinter-info"></span>
  <span class="status-hinter-warning"></span>
  <span class="status-hinter-attention"></div>
</div>
```

## Sizings
Set status hinter size using the `.status-hinter-[size]` utility class or `data-size="[size]"` attribute. Options available are `sm`, `md`, `lg`. Usage example is `status-hinte-lg` or `data-size="lg"`.

```html transparent
<template preview>
  <div>
    <span class="status-hinter" data-size="sm"></span>
    <span class="status-hinter-success" data-size="sm"></span>
    <span class="status-hinter-danger" data-size="sm"></span>
    <span class="status-hinter-primary" data-size="sm"></span>
    <span class="status-hinter-info" data-size="sm"></span>
    <span class="status-hinter-warning" data-size="sm"></span>
    <span class="status-hinter-attention" data-size="sm"></div>
  </div>

  <div>
    <span class="status-hinter" data-size="md"></span>
    <span class="status-hinter-success" data-size="md"></span>
    <span class="status-hinter-danger" data-size="md"></span>
    <span class="status-hinter-primary" data-size="md"></span>
    <span class="status-hinter-info" data-size="md"></span>
    <span class="status-hinter-warning" data-size="md"></span>
    <span class="status-hinter-attention" data-size="md"></div>
  </div>

  <div>
    <span class="status-hinter" data-size="lg"></span>
    <span class="status-hinter-success" data-size="lg"></span>
    <span class="status-hinter-danger" data-size="lg"></span>
    <span class="status-hinter-primary" data-size="lg"></span>
    <span class="status-hinter-info" data-size="lg"></span>
    <span class="status-hinter-warning" data-size="lg"></span>
    <span class="status-hinter-attention" data-size="lg"></div>
  </div>
  
</template>

<div>
  <span class="status-hinter" data-size="sm"></span>
  <span class="status-hinter" data-size="md"></span>
  <span class="status-hinter" data-size="lg"></span>
</div>
```
