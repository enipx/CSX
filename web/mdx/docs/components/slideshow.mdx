## Usage

```html
<template preview>
  <!-- --------- Slideshow ----------- -->
  <div id="slideshow1" class="slideshow mb-8 mt-2 mx-auto" style="max-width: 50%" data-slide-animation="fade">
    <div class="slideshow_indicators" data-pos='offset' data-variant="">
      <span data-slideshow="#slideshow1" data-slide-to="1" class="active"></span>
      <span data-slideshow="#slideshow1" data-slide-to="2"></span>
      <span data-slideshow="#slideshow1" data-slide-to="3"></span>
    </div>
  
    <div class="slideshow_content px-10 py-4">
      <div class="slideshow_item active">
        <h4>Testimonial One</h4>
        <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Consequatur laboriosam, magnam nemo perspiciatis error magni iusto necessitatibus ratione expedita a!</p>
      </div>
  
      <div class="slideshow_item">
        <h4>Testimonial Two</h4>
        <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Consequatur laboriosam, magnam nemo perspiciatis error magni iusto necessitatibus ratione expedita a!</p>
      </div>
  
      <div class="slideshow_item">
        <h4>Testimonial Three</h4>
        <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Consequatur laboriosam, magnam nemo perspiciatis error magni iusto necessitatibus ratione expedita a!</p>
      </div>
    </div>
  
    <div class="slideshow_control">
      <button class="slideshow_prev" data-slideshow="#slideshow1" data-slide-to="prev">
        <span class="slideshow_prev_icon"></span>
      </button>
      <button class="slideshow_next" data-slideshow="#slideshow1" data-slide-to="next">
        <span class="slideshow_next_icon"></span>
      </button>
    </div>
  </div>
</template>

<div>

      <div class="slideshow_item">
        <h4>Testimonial Two</h4>

      </div>

</div>
```

## Auto Slide
Use the `data-auto-slide` attribute to allow auto slide

```html
<template preview>
  <!-- --------- Slideshow ----------- -->
  <div id="slideshow2" class="slideshow mb-8 mt-2 mx-auto" style="max-width: 50%" data-auto-side data-slide-duration="" data-slide-animation="fade">
    <div class="slideshow_indicators" data-pos='offset' data-variant="">
      <span data-slideshow="#slideshow2" data-slide-to="1" class="active"></span>
      <span data-slideshow="#slideshow2" data-slide-to="2"></span>
      <span data-slideshow="#slideshow2" data-slide-to="3"></span>
    </div>
  
    <div class="slideshow_content px-10 py-4">
      <div class="slideshow_item active">
        <h4>Testimonial One</h4>
        <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Consequatur laboriosam, magnam nemo perspiciatis error magni iusto necessitatibus ratione expedita a!</p>
      </div>
  
      <div class="slideshow_item">
        <h4>Testimonial Two</h4>
        <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Consequatur laboriosam, magnam nemo perspiciatis error magni iusto necessitatibus ratione expedita a!</p>
      </div>
  
      <div class="slideshow_item">
        <h4>Testimonial Three</h4>

  </div>
</template>

```
