## Usage

```html transparent
<template preview>
  <button type="button" class="icon-btn">
    <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="icon-btn_icon">
      <line x1="5" y1="12" x2="19" y2="12"></line>
      <polyline points="12 5 19 12 12 19"></polyline>
    </svg>
  </button>
</template>

<div>
  <button type="button" class="icon-btn">
    <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" class="icon-btn_icon">
      <line x1="5" y1="12" x2="19" y2="12"></line>
      <polyline points="12 5 19 12 12 19"></polyline>
    </svg>
  </button>
</div>
```

## Sizings
Set button size using the `.icon-btn-[size]` utility class or `data-size="[size]"` attribute. Options available are `xs`, `sm`, `md`, `lg`, `xl`. Usage example is `icon-btn-lg` or `data-size="lg"`.

## Custom Usage
You can combine utilities to get a desired effect.

```html transparent
<template preview>
  <div class="d-flex">
    <button type="button" class="m-2 icon-btn bdrs-full" data-size="xl">
      <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="icon-btn_icon">
        <line x1="5" y1="12" x2="19" y2="12"></line>
        <polyline points="12 5 19 12 12 19"></polyline>
      </svg>
    </button>

    <button type="button" class="m-2 icon-btn bg-primary hover-bg-primary-600 c-white" data-size="xl">
      <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="icon-btn_icon">
        <line x1="5" y1="12" x2="19" y2="12"></line>
        <polyline points="12 5 19 12 12 19"></polyline>
      </svg>
    </button>

    <button type="button" class="m-2 icon-btn bg-grad-orange c-white" data-size="xl">
      <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="icon-btn_icon">
        <line x1="5" y1="12" x2="19" y2="12"></line>
        <polyline points="12 5 19 12 12 19"></polyline>
      </svg>
    </button>
  </div>
</template>

<div>
  <button type="button" class="icon-btn bdrs-full">
      ....
  </button>

  <button type="button" class="icon-btn bg-primary hover-bg-primary-600">
      ....
  </button>

  <button type="button" class="icon-btn bg-grad-orange">
      ....
  </button>
</div>
```
