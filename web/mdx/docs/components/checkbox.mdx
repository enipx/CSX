## Usage

```html
<template preview>
  <div>
    <input type="checkbox" class="checkbox" id="male" />
    <label for="male">Male</label>
  </div>
</template>

<div>
  <div>
    <input type="checkbox" class="checkbox" id="male" />
    <label for="male">Male</label>
  </div>
</div>
```

## Sizings
Set textarea size using the `.checkbox-[size]` utility class or `data-size="[size]"` attribute. Options available are `sm`, `md`, `lg`. Usage example is `checkbox-lg` or `data-size="lg"`.

```html
<template preview>
  <div class="d-flex">
    <div class="mr-4">
      <input type="checkbox" class="checkbox checkbox-sm" id="male-sm" />
      <label for="male-sm">Male</label>
    </div>

    <div class="mr-4">
      <input type="checkbox" class="checkbox checkbox-md" id="male-md" />
      <label for="male-md">Male</label>
    </div>

    <div>
      <input type="checkbox"  class="checkbox checkbox-lg" id="male-lg" />
      <label for="male-lg">Male</label>
    </div>
  </div>
</template>

<div>
  <div>
    <input type="checkbox" class="checkbox checkbox-sm" id="male-sm" />
    <label for="male-sm">Male</label>
  </div>

  <div>
    <input type="checkbox" class="checkbox" id="male-md" />
    <label for="male-md">Male</label>
  </div>

  <div>
    <input type="checkbox" class="checkbox checkbox-lg" id="male-lg" />
    <label for="male-lg">Male</label>
  </div>
</div>
```

## States
Update textarea state by using 
 - `data-invalid='true'`/`aria-invalid="true"` attribute to indicate an invalid state, 
 - `checked` attribute to indicate a checked state,
 - `disabled` attribute to indicate a disabled state,

```html
<template preview>
  <div class="">
    <div>
      <input type="checkbox" class="checkbox" id="checkbox-invalid" aria-invalid="true" />
      <label for="checkbox-invalid">Invalid</label>
    </div>

    <div>
      <input type="checkbox" class="checkbox" id="checkbox-checked" checked />
      <label for="checkbox-checked">Checked</label>
    </div>

    <div>
      <input type="checkbox" class="checkbox" id="checkbox-disabled" disabled />
      <label for="checkbox-disabled">Disabled</label>
    </div>
  </div>
</template>

<div>
  <div>
    <input type="checkbox" class="checkbox" id="checkbox-invalid" aria-invalid="true" />
    <label for="checkbox-invalid">Invalid</label>
  </div>

  <div>
    <input type="checkbox" class="checkbox" id="checkbox-disabled" checked disabled />
    <label for="checkbox-disabled">Disabled</label>
  </div>
</div>
```
