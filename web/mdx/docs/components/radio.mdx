---
title: "Radio"
description: "Used to add options when only a single option can be selected from the given options."
---

## Usage

```html
<template preview>
  <div>
    <input type="radio" class="radio" id="male" />
    <label for="male">Male</label>
  </div>
</template>

<div>
  <div>
    <input type="radio" class="radio" id="male" />
    <label for="male">Male</label>
  </div>
</div>
```

## Sizings
Set radio size using the `.radio-[size]` utility class or `data-size="[size]"` attribute. Options available are `sm`, `md`, `lg`. Usage example is `radio-lg` or `data-size="lg"`.

```html
<template preview>
  <div class="d-flex">
    <div class="mr-4">
      <input type="radio" class="radio radio-sm" name="gender" id="male-sm" />
      <label for="male-sm">Male</label>
    </div>

    <div class="mr-4">
      <input type="radio" class="radio" name="gender" id="female-md" />
      <label for="female-md">Female</label>
    </div>

    <div>
      <input type="radio" class="radio radio-lg" name="gender" id="other-lg" />
      <label for="other-lg">Other</label>
    </div>
  </div>
</template>

<div>
  <div>
    <input type="radio" class="radio radio-sm" name="gender" id="male-sm" />
    <label for="male-sm">Male</label>
  </div>

  <div>
    <input type="radio" class="radio" name="gender" id="female-md" />
    <label for="female-md">Female</label>
  </div>

  <div>
    <input type="radio" class="radio radio-lg" name="gender" id="other-lg" />
    <label for="other-lg">Other</label>
  </div>
</div>
```

## States
Update textarea state by using 
 - `data-invalid`/`aria-invalid="true"` attribute to indicate an invalid state, 
 - `disabled` attribute to indicate a disabled state,
 - `checked` attribute to indicate a checked state,

```html
<template preview>
  <div class="">
    <div class="mb-4">
      <input type="radio" class="radio" name="state" id="radio-invalid" aria-invalid="true" />
      <label for="radio-invalid">Invalid</label>
    </div>

    <div class="mb-4">
      <input type="radio" name="state" class="radio" id="radio-disabled" disabled />
      <label for="radio-disabled">Disabled</label>
    </div>

    <div class="mb-4">
      <input type="radio" name="state" class="radio" id="radio-checked" checked />
      <label for="radio-checked">Checked</label>
    </div>
  </div>
</template>

<div>
  <div>
    <input type="radio" class="radio" name="state" id="radio-invalid" aria-invalid="true" />
    <label for="radio-invalid">Invalid</label>
  </div>

  <div>
    <input type="radio" class="radio" name="state" id="radio-disabled" checked disabled />
    <label for="radio-disabled">Disabled</label>
  </div>
</div>
```
